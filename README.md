## Style guide

### Общее по тексту

### Раздел «История изменений». Как оформлять?
- заголовок 2 уровня → Месяц и год изменения
- заголовок 3 уровня → Классификация изменения:

  - Новый функционал
  - Исправление ошибок
  - Улучшение
